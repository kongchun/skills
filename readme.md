# AI Skills Collection (灵码技能库)

本项目是一个模块化的 AI 智能体技能（Skills）收集仓库。旨在为 AI 助手（如阿里云灵码 Lingma）提供垂直领域的专业知识、标准化工作流及自动化工具脚本，从而增强其在特定任务下的执行能力与准确性。

## 📂 已收录技能

| 技能名称 | 描述 | 触发关键词 |
| :--- | :--- | :--- |
| [junior-study-notes](./junior-study-notes/) | **初中教材知识要点整理**<br>支持苏科版/人教版等主流教材。自动生成教材目录、Markdown 知识点笔记及可打印 Word 文档。覆盖语数英物化生史地政全学科。 | `初中教材`、`知识要点`、`七年级/八年级/九年级`、`复习规划`、`教材目录` |

## 🚀 快速开始

### 1. 使用技能
在对话中直接提及技能对应的**触发关键词**，或明确指定技能名称，AI 助手将自动加载对应的 [[SKILL.md](junior-study-notes\SKILL.md)](junior-study-notes\SKILL.md) 及关联资源。

**示例：**
> "帮我整理苏科版八年级物理上册的知识要点，并生成 Word 文档。"

### 2. 技能结构规范
每个技能文件夹应包含以下结构：

```text
{skill-name}/
├── SKILL.md               # [必选] 技能核心定义、工作流、Prompt 模板
├── references/            # [可选] 静态知识库、API 文档、避坑指南
└── scripts/               # [可选] 自动化处理脚本 (Python/JS 等)
