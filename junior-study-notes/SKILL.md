# 初中教材知识要点整理

## 工作流程

### 第一步：确认教材版本

1. 向用户确认以下信息（缺一不可）：
   - **科目**：物理、生物、数学、语文、英语、化学、地理、历史、政治…
   - **年级 + 学期**：七年级上/下、八年级上/下、九年级上/下
   - **出版社**：苏科版（江苏凤凰科学技术出版社）、苏教版（江苏教育出版社）、人教版（人民教育出版社）、北师大版等
2. 若用户不确定出版社，查 `references/publishers.md` 中苏州/江苏常见教材对照表辅助判断
3. **务必先输出目录让用户确认**，再生成详细内容，避免版本错误返工

### 第二步：获取教材目录

按优先级尝试：

1. **抓取 dzkbw.com**：参考 `references/dzkbw-guide.md` 的 URL 规律和编码处理方法
2. **搜索引擎**：用 Toutiao / Bing 搜索「{出版社}+{年级}+{科目}+上册+教材目录」
3. **用户提供**：请用户拍照或手写目录

> ⚠️ 抓取 dzkbw.com 时注意：网站有反爬（403），需 SSL 禁用验证 + 完整浏览器 UA；页面编码 GBK，需 `html.decode('gbk')`。详见 `references/dzkbw-guide.md`

### 第三步：生成 Markdown 知识要点

每章生成一个 `.md` 文件，结构如下：

```markdown
# 第X章  章名

## 本章学习目标
| 目标 | 内容 |
|:---|:---|
| 理解 | ... |
| 掌握 | ... |

## X.1  节名
### 核心概念/公式
### 典型例题
### 易错警示（⚠️）

## X.2  节名
...

## ⭐ 本章核心公式速查
## ⚠️ 本章易错点
```

文件命名：`第X章_章名.md`

写入路径：`%USERPROFILE%\Desktop\初中教材\{年级}{科目}{学期}\`

### 第四步：生成 Word 文档（可选）

用户要求时，用 docx-js 生成可打印 Word 文档：

1. 读取 `scripts/gen_word.js` 模板
2. 按模板填充各章内容
3. **若科目为语文**，需额外读取 `必背重点词语_古诗文.md` 并追加为最后一个章节（放在单元章节之后、附录之前）
4. 输出 `.docx` 文件

> ⚠️ JS 中文引号陷阱：JS 字符串中的中文引号 `""` 会被解析为字符串边界导致 SyntaxError，必须替换为角括号 `「」` 或英文引号。详见 `references/js-pitfalls.md`

### 第五步：清理临时文件

- 删除所有 Python/JS 中间脚本
- 保留最终 .md 和 .docx 文件

## 文件编码规则

- **Markdown 文件**：UTF-8（无 BOM）
- **Word 文件**：docx-js 自动处理
- **Python 脚本**：首行 `# -*- coding: utf-8 -*-`
- **JS 脚本**：保存为 UTF-8
- **终端输出**：避免 emoji，用 `[OK]` 替代 ✓，避免 GBK 终端报错

## 各科目知识要点覆盖模板

### 通用板块（所有科目）

| 板块 | 内容 |
|:---|:---|
| 学习目标 | 理解/掌握/会用的分级目标 |
| 核心概念 | 定义、特点、分类（表格化） |
| 典型例题 | 题目 + 解题思路 + 答案 |
| 易错警示 | ⚠️ 常见错误 + 正确理解 |
| 章节对比 | 易混淆概念对比表 |

### 理科专属板块（数学、物理、化学）

| 板块 | 内容 |
|:---|:---|
| 公式定理 | 公式 + 变形公式表 |
| 实验步骤 | 编号列表，关键操作加粗（物理/化学） |
| 公式速查 | 本章所有公式一览 |

### 文科专属板块（语文、历史、地理、道德与法治）

| 板块 | 内容 |
|:---|:---|
| 重点字词/词汇 | 语文：字音字形、词语解释 |
| 时间轴/事件线 | 历史：重大事件时间线 |
| 地图要素 | 地理：区域特征、气候类型、地形分布 |
| 观点论述 | 道法：核心观点 + 论据 + 答题模板 |

### 英语专属板块

| 板块 | 内容 |
|:---|:---|
| 核心词汇 | 单词表 + 词性 + 例句 |
| 语法要点 | 语法规则 + 例句 + 对比辨析 |
| 句型模板 | 常用句型 + 写作套用 |

### 生物专属板块

| 板块 | 内容 |
|:---|:---|
| 结构图解 | 细胞/器官结构名称与功能 |
| 生命活动 | 光合作用/呼吸作用等过程表格 |
| 实验方案 | 实验目的、材料、步骤、结论 |

### 语文专属板块（含古诗文）

每章 Markdown 包含：

| 板块 | 内容 |
|:---|:---|
| 单元简介 | 主题 + 学习方向 |
| 课文详解 | 主旨 + 重点句段（原文+赏析）+ 写作手法 |
| 古诗文原文 | 原文逐句（含注音）+ 注释 + 翻译 + 赏析 |
| 古诗文名句 | 名句 + 名句默写考题 |
| 写作指导 | 写作要点 + 技巧 + 范文示例 |

必背古诗文单独生成一份 `必背重点词语_古诗文.md`，包含：
- 按单元分栏的**所有重点词语表**（字音/字形/释义/例句）
- 所有古诗文**原文逐句**（含注音）+ 重点注释 + 全文翻译
- **名句名句默写**，标记常考句
- **文学常识速查表**（作者/出处/朝代）
- **易考题型归纳**（文学常识题/默写题/文言文阅读理解题答题格式）

> **语文特殊性**：古诗文是高频考点，必须逐字逐句整理。课文详解中重点句段的赏析要写出修辞手法和表达效果。
